RCSS is the version of Re-Console specifically meant to be ran on servers. It comes with more performance improvements then just running default Re-Console on a server, whilst being much more streamlined.



## 📥 Installation Guide
There are multiple ways to install the Re-Console server software.

<details>
<summary>Methods</summary>

<details>
<summary>mrpack install</summary>
Download mrpack-install through GitHub releases (or your distro's package if it has one) and take a look at the commands on the README. In Re-Console's case, to install in your server you would run:

``mrpack-install re-console-server-software [optional version number]``



</details>

<details>
<summary>mrpack4server</summary>
See the mrpack4server Git repository for installation info. modpack-info.json:

``{
  "project_id": ",
  "version_id": "version id or name"
}``

</details>
